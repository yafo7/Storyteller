# 美术素材清单 — list.md

> 提交截止：2026.06.20
> 风格方向：2D 俯视角，暗色基调，哥特/维多利亚氛围

---

## 一、NPC 立绘与头像

每位 NPC 需要：**全身立绘（可选，对话框使用）+ 方形头像（128×128）**。

---

### 1. Anne Stewart — 长女 · 通灵者

| 属性 | 描述 |
|------|------|
| **年龄** | 约 28-32 岁，成年女性 |
| **外貌** | 肤色苍白，深棕色卷发披肩，眼眶微红（常哭泣），穿暗色维多利亚长裙 |
| **表情** | 忧郁、神经质，眼角总是湿润 |
| **头像要求** | 侧光打脸，半明半暗，眼神看向远方（仿佛在看不可见之物） |
| **色调** | 暖粉灰 #B482A0 为基底 |
| **参考** | 成年版 Nicole Kidman 饰演的 Anne |

---

### 2. Nicholas Stewart — 幼子 · 沉默者

| 属性 | 描述 |
|------|------|
| **年龄** | 约 26-30 岁，成年男性 |
| **外貌** | 瘦削，短黑发，总是低头或避开眼神接触，穿深灰色西装马甲 |
| **表情** | 恐惧、回避，偶尔流露儿童般的脆弱 |
| **头像要求** | 低头，半张脸在阴影中，手指紧握圣经或念珠 |
| **色调** | 冷灰蓝 #8CA0C8 |
| **参考** | 被战争和死亡吓坏的男孩长大后的样子 |

### 3. Charles Stewart — 父亲 · 归来的战士

| 属性 | 描述 |
|------|------|
| **年龄** | 约 42-48 岁 |
| **外貌** | 军装（一战英军风格）已破旧，脸上有伤疤，眼神空洞，胡茬 |

| **色调** | 沙棕 #A08C6E |

## 二、场景素材

### 会客厅背景

| 元素 | 描述 |
|------|------|
| **整体** | 维多利亚风格会客厅，暗色调，烛光照明 |
| **墙壁** | 深色木饰面板（wainscoting），上部是褪色的花纹墙纸 |
| **地板** | 深色硬木地板，中央铺波斯地毯（红色褪成暗棕） |
| **天花板** | 一盏水晶吊灯，无电，蜡烛代替灯泡 |
| **壁炉** | 石砌壁炉，内有微弱火光，上方挂一面大镜子（雾蒙蒙） |
| **窗户** | 三扇高窗，窗帘全部被扯掉，外面是浓雾 |

### 中央长桌

| 元素 | 描述 |
|------|------|
| **桌子** | 深色桃花心木长桌，可坐 8-10 人 |
| **桌面物品** | 复古烛台 × 3，老照片册一本，茶杯若干 |
| **椅子** | 6 把高背椅，暗红丝绒坐垫 |

### 其他装饰

| 元素 | 描述 |
|------|------|
| **书架** | 靠墙，摆满旧书，几本散落在地 |
| **钢琴** | 角落的旧钢琴，琴盖合上，积灰 |
| **地毯** | 波斯风格，深红/暗金，边缘磨损 |
| **蜡烛** | 多个烛台散落，火焰为唯一动态光源 |

## 三、UI 元素

| 元素 | 描述 |
|------|------|
| **[E] Talk 提示** | 半透明金底，渐隐渐显 |
| **对话气泡** | NPC 头顶白色圆角气泡，文字深棕，2 秒后渐隐 |
| **NPC 对话面板** | 底部 700px 宽的暗色半透明面板，NPC 名字左侧 + 头像 + 文字 + 3 个选项 |
| **旁白区域** | 屏幕底部中央，淡入淡出，白色衬线字体 |
| **表情图标** | NPC 头顶小号 emoji 表示当前情绪（😨 😢 😠 😶 🕯️） |

## 四、音效建议（可选）

| 场景 | 建议 |
|------|------|
| **环境** | 壁炉噼啪声 + 远处风声 |
