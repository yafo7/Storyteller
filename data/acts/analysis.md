# 剧本分析报告 — 《小岛惊魂》

## 主题识别

这个故事本质上是关于**死亡认知与身份反转**。
所有人都生活在某种自我欺骗中——Grace 认为自己是受害者，实际上她才是加害者；
活人以为房子闹鬼，实际上他们才是"入侵者"。
核心命题：**"我们如何知道自己还活着？"**

## 叙事结构

线性时间推进 + 最后一幕反转颠覆所有前提。
所有铺垫（窗帘、光过敏、失踪的仆人、"鬼魂"）在反转后获得截然不同的含义。

## 反转点

**反转前世界**：Grace 是可怜的母亲，死去的丈夫，恐惧的孩子，被神秘的"其他人"骚扰。
**反转后世界**：Grace 闷死了两个孩子后自杀。她们才是鬼魂。活人是新搬来的 Marlish 一家。

## 情绪曲线

```
疑惑 → 压抑 → 恐惧 → 崩溃 → 反转（认知崩塌）→ 释然
```

## 叙事技法

- **不可靠叙事**：Grace 视角全程误导观众
- **多视角**：Anne 能看到真相、Mills 知道真相但不说不说
- **信息遮蔽**：观众和 Grace 同步接收信息，一起被欺骗
- **反转**：所有前提在最后一刻全部颠覆
- **象征**：雾 = 死亡与遗忘；光 = 真相与痛苦；窗帘 = 自欺欺人
- **沉默叙事**：Lydia 的哑 = 真相的不可言说

## 建议分幕：8 幕

0. 序章 — 受邀而至
1. 雾中宅邸 — 世界建立
2. 孩子的秘密 — Victor 出现
3. 入侵者 — 窗帘消失
4. 死者之书 — 相册揭示
5. 三座墓碑 — 真相逼近
6. 真相 — 反转揭示 + Marlish 一家进入
7. 此屋归我 — 收束结局
